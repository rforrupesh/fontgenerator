Cool WhatsApp Chat: Text Formatting, Fonts, and Shortcut Hacks

1. Text Formatting: Make Your Messages Stand Out

WhatsApp allows you to format your text in bold, italics, strikethrough, and monospace. Here’s how:

Bold Text

Use an asterisk (*) before and after the text.

Example: Type *Hello* and it will appear as Hello.

Italic Text

Use an underscore (_) before and after the text.

Example: Type _How are you?_ and it will appear as How are you?.

Strikethrough Text

Use a tilde (~) before and after the text.

Example: Type ~This is old~ and it will appear as This is old.

Monospace Text

Use three backticks (```) before and after the text.

Example: Type ```Check this out``` and it will appear as Check this out.

3. Shortcut Hacks: Save Time While Chatting

Here are some useful WhatsApp shortcuts:

Quick Replies

Type a shortcut word, and WhatsApp will suggest your saved reply.

Keyboard Shortcuts

  • Ctrl + Enter (Windows) / Command + Enter (Mac): Send your message instantly.
  • Ctrl + Shift + \ (Windows) / Command + Shift + \ (Mac): Switch between chats quickly.

Starred Messages

Bookmark important messages by long-pressing them and tapping the star icon.

4. Bonus Tips for Better WhatsApp Experience

  • Pin Important Chats: Long-press a chat and select the pin icon to keep it at the top.
  • Use WhatsApp Web: Access WhatsApp on your computer for easier typing.
  • Mute Group Chats: Long-press a group chat and select “Mute Notifications” to silence it.

5. Conclusion: Spice Up Your Chats Today!

By using text formatting, custom fonts, and shortcuts, you can make your WhatsApp chats more engaging and efficient. Try these hacks today and impress your friends!
